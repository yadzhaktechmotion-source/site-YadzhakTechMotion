# Security

Security is a first-class concern in this project.

---

## Transport Security

- HTTPS enforced
- HSTS enabled
- Secure cookies

---

## Authentication

- Passwordless email-code login
- Rate-limited login requests
- Session expiration policies

---

## Application Security

- CSRF protection
- Clickjacking protection
- Secure HTTP headers
- Dependency updates

---

## Secrets Management

- Docker secrets
- No secrets in Git
- Per-environment isolation

---

## Email Anti-Spoofing

- SPF
- DKIM
- DMARC (enforced)
- BIMI-ready configuration
